Quiz game rules

  • Each question has 7 seconds.

  • Select only one answer per question.

  • Correct answers earn 1 point.

  • Wrong answer earn 0 point.

  • If time runs out, the question is skipped.

  • Do not refresh the page during game.

  • Your final score will be shown at the end.

  • The top ten highest scorer are saved on the leaderboard.

  • Click start to start the game.
